## Task 1: PhotonVision Subsystem (NEW FILE)

### File: `src/main/java/frc/robot/subsystems/PhotonVisionSubsystem.java`

A new `SubsystemBase` that:
- Instantiates two `PhotonCamera` objects (`"photon-right"`, `"photon-left"`)
- Defines `Transform3d` placeholder constants for each camera's robot-to-camera mount
- Periodically queries `PhotonPoseEstimator` for each camera's best pose estimate
- Feeds valid measurements into YAGSL's pose estimator via `swerveDrive.addVisionMeasurement()`

```java
package frc.robot.subsystems;

import edu.wpi.first.math.geometry.*;
import edu.wpi.first.wpilibj2.command.SubsystemBase;
import org.photonvision.PhotonCamera;
import org.photonvision.PhotonPoseEstimator;
import org.photonvision.EstimatedRobotPose;
import frc.robot.Constants.VisionConstants;
import frc.robot.subsystems.swervedrive.SwerveSubsystem;

public class PhotonVisionSubsystem extends SubsystemBase {

    // Two PhotonVision cameras (names must match PhotonVision UI config)
    private final PhotonCamera m_rightCamera = new PhotonCamera("photon-right");
    private final PhotonCamera m_leftCamera  = new PhotonCamera("photon-left");

    // Pose estimators (one per camera, uses fixed tag map for field-relative pose)
    private final PhotonPoseEstimator m_rightEstimator;
    private final PhotonPoseEstimator m_leftEstimator;

    private final SwerveSubsystem m_drivebase;

    public PhotonVisionSubsystem(SwerveSubsystem drivebase) {
        m_drivebase = drivebase;

        // Create estimators: constructor takes field layout + camera-to-robot transform
        // Strategy is set per-call via which estimate* method we invoke
        m_rightEstimator = new PhotonPoseEstimator(
            VisionConstants.APRIL_TAG_FIELD_LAYOUT,
            VisionConstants.RIGHT_CAMERA_TRANSFORM
        );
        m_leftEstimator = new PhotonPoseEstimator(
            VisionConstants.APRIL_TAG_FIELD_LAYOUT,
            VisionConstants.LEFT_CAMERA_TRANSFORM
        );
    }

    @Override
    public void periodic() {
        // Query each camera and feed to swerve pose estimator
        updateFromCamera(m_rightCamera, m_rightEstimator);
        updateFromCamera(m_leftCamera, m_leftEstimator);
    }

    private void updateFromCamera(PhotonCamera camera, PhotonPoseEstimator estimator) {
        // getAllUnreadResults() is preferred over deprecated getLatestResult()
        var results = camera.getAllUnreadResults();
        for (var result : results) {
            if (!result.hasTargets()) continue;

            // Try multi-tag PnP on coprocessor first (most accurate)
            var poseOptional = estimator.estimateCoprocMultiTagPose(result);

            // Fallback to lowest ambiguity if multi-tag unavailable
            if (poseOptional.isEmpty()) {
                poseOptional = estimator.estimateLowestAmbiguityPose(result);
            }

            if (poseOptional.isPresent()) {
                EstimatedRobotPose poseEstimate = poseOptional.get();
                // Feed to YAGSL's pose estimator
                m_drivebase.getSwerveDrive().addVisionMeasurement(
                    poseEstimate.estimatedPose,
                    poseEstimate.timestampSeconds
                );
            }
        }
    }

    /** Check if a specific camera sees a specific AprilTag ID. */
    public boolean hasTarget(PhotonCamera camera, int tagId) {
        var results = camera.getAllUnreadResults();
        for (var result : results) {
            if (!result.hasTargets()) continue;
            for (var target : result.getTargets()) {
                if (target.getFiducialId() == tagId) return true;
            }
        }
        return false;
    }

    /** Get yaw to a specific tag from a specific camera, or NaN if not found. */
    public double getTargetYaw(PhotonCamera camera, int tagId) {
        var results = camera.getAllUnreadResults();
        for (var result : results) {
            if (!result.hasTargets()) continue;
            for (var target : result.getTargets()) {
                if (target.getFiducialId() == tagId) {
                    return target.getYaw();  // degrees, CCW+
                }
            }
        }
        return Double.NaN;
    }

    /** Get a camera by name for use in commands. */
    public PhotonCamera getCamera(String name) {
        if (name.equals("right")) return m_rightCamera;
        return m_leftCamera;
    }
}
```

## Task 2: ID-Specific Auto-Aim with Odometry Fallback (NEW COMMAND)

### File: `src/main/java/frc/robot/commands/AutoAimWithFallbackCommand.java`

New command that follows the priority list:
1. Check if a Reef tag ID is visible in vision
2. If yes → use vision yaw for turret correction
3. If no → fallback to odometry angle to Reef center coordinates

```java
package frc.robot.commands;

import edu.wpi.first.math.geometry.Pose2d;
import edu.wpi.first.math.geometry.Translation2d;
import edu.wpi.first.wpilibj.DriverStation;
import edu.wpi.first.wpilibj2.command.Command;
import frc.robot.Constants.VisionConstants;
import frc.robot.subsystems.TurretSubsystem;
import frc.robot.subsystems.PivotSubsystem;
import frc.robot.subsystems.FlywheelSubsystem;
import frc.robot.subsystems.swervedrive.SwerveSubsystem;
import frc.robot.subsystems.PhotonVisionSubsystem;
import frc.robot.util.VisionUtil;

/**
 * Auto-aim with vision→odometry fallback:
 * 1. Primary: if any alliance hub tag visible → use vision yaw
 * 2. Fallback: if no hub tag visible → compute angle from odometry to alliance hub
 */
public class AutoAimWithFallbackCommand extends Command {

    private final SwerveSubsystem m_drivebase;
    private final TurretSubsystem m_turret;
    private final PivotSubsystem m_pivot;
    private final PhotonVisionSubsystem m_photonVision;

    // Which camera to use for primary aiming (configurable)
    private final String m_cameraName;

    public AutoAimWithFallbackCommand(SwerveSubsystem drive, TurretSubsystem turret,
                                       PivotSubsystem pivot,
                                       PhotonVisionSubsystem photonVision, String cameraName) {
        m_drivebase = drive;
        m_turret = turret;
        m_pivot = pivot;
        m_photonVision = photonVision;
        m_cameraName = cameraName;
        addRequirements(turret, pivot);
    }

    @Override
    public void execute() {
        int[] hubTagIds = getHubTagIds();
        var camera = m_photonVision.getCamera(m_cameraName);

        // === PRIORITY 1: Vision — check if any hub tag is visible ===
        for (int tagId : hubTagIds) {
            if (m_photonVision.hasTarget(camera, tagId)) {
                double yawDeg = m_photonVision.getTargetYaw(camera, tagId);
                if (!Double.isNaN(yawDeg)) {
                    // Vision sees a hub tag — use yaw for turret correction
                    // yawDeg is angular offset from camera center (degrees, CCW+)
                    double robotHeading = m_drivebase.getHeading().getRadians();
                    double cameraYawRad = Math.toRadians(yawDeg);
                    // Field angle = robot heading + camera yaw
                    double fieldAngle = robotHeading + cameraYawRad;
                    double turretAngle = VisionUtil.computeTurretAngle(fieldAngle, robotHeading);
                    m_turret.setTargetAngle(turretAngle);
                    return;
                }
            }
        }

        // === PRIORITY 2: Odometry Fallback — compute angle to alliance hub ===
        Pose2d robotPose = m_drivebase.getPose();
        Translation2d hubPos = getHubPosition();
        double fieldAngle = VisionUtil.computeOdometryAngleToTarget(robotPose, hubPos);
        double robotHeading = m_drivebase.getHeading().getRadians();
        double turretAngle = VisionUtil.computeTurretAngle(fieldAngle, robotHeading);
        m_turret.setTargetAngle(turretAngle);
    }

    @Override
    public void end(boolean interrupted) {
        m_turret.setTargetAngle(0.0);
        m_pivot.setTargetAngle(0.0);
    }

    /** Get the hub tag IDs based on current alliance. */
    private int[] getHubTagIds() {
        var alliance = DriverStation.getAlliance();
        if (alliance.isPresent() && alliance.get() == DriverStation.Alliance.Red) {
            return VisionConstants.RED_TAG_IDS;
        }
        return VisionConstants.BLUE_TAG_IDS;
    }

    /** Get the hub field position based on current alliance. */
    private Translation2d getHubPosition() {
        var alliance = DriverStation.getAlliance();
        if (alliance.isPresent() && alliance.get() == DriverStation.Alliance.Red) {
            return VisionConstants.HUB_POSITION_RED;
        }
        return VisionConstants.HUB_POSITION_BLUE;
    }
}
```

## Task 3: Turret Angle Math (ADD TO VisionUtil)

### File: `src/main/java/frc/robot/util/VisionUtil.java` — Add two methods

```java
// Required imports (add to VisionUtil.java):
// import edu.wpi.first.math.MathUtil;
// import edu.wpi.first.math.geometry.Pose2d;
// import edu.wpi.first.math.geometry.Translation2d;

/**
 * Compute field-relative angle from robot to a target using odometry.
 *
 * @param robotPose  Current robot pose from swerve odometry
 * @param targetPos  Field coordinates of the target (e.g., alliance hub)
 * @return Field-relative angle (radians, CCW+) from robot to target
 */
public static double computeOdometryAngleToTarget(Pose2d robotPose, Translation2d targetPos) {
    Translation2d robotToTarget = targetPos.minus(robotPose.getTranslation());
    return robotToTarget.getAngle().getRadians();
}

/**
 * Convert field-relative target angle to turret encoder angle.
 *
 * Coordinate math (WPILib CCW+):
 *   1. fieldAngle: 0° = +x (forward), 90° = +y (left)
 *   2. robotHeading: 0° = facing +x
 *   3. robotRelative = fieldAngle - robotHeading    (field → robot frame)
 *   4. turretEncoder = robotRelative - π             (offset: encoder 0° = robot back)
 *   5. Wrapped to [-π, π] via inputModulus
 *
 * @param fieldAngle    Target angle in field frame (radians, CCW+)
 * @param robotHeading  Current robot heading (radians, CCW+)
 * @return Turret encoder target angle (degrees, for TurretSubsystem.setTargetAngle)
 */
public static double computeTurretAngle(double fieldAngle, double robotHeading) {
    // Step 1: field-relative → robot-relative
    double robotRelative = fieldAngle - robotHeading;
    // Step 2: apply 180° offset (encoder 0° = back of robot)
    double turretRadians = robotRelative - Math.PI;
    // Step 3: wrap to [-π, π] — shortest path, no infinite winding
    turretRadians = MathUtil.inputModulus(turretRadians, -Math.PI, Math.PI);
    // Step 4: convert to degrees for TurretSubsystem
    return Math.toDegrees(turretRadians);
}
```

**Verification of formula:**

| Robot Heading | Target (field) | Robot-Rel | Turret Encoder | Faces |
|---|---|---|---|---|
| 0° | 180° (behind) | 180° | 0° | Back ✓ |
| 20° | 180° (behind) | 160° | -20° | Compensates ✓ |
| 0° | 90° (left) | 90° | -90° | Left ✓ |
| 0° | 0° (front) | 0° | -180°→180° | Front ✓ |
