## Task 1: Combined AutoAim + AutoShoot (Right Trigger)

### 1A. Create `AutoAimAndShootCommand.java`
**File:** `src/main/java/frc/robot/commands/AutoAimAndShootCommand.java` (NEW)

Combines AutoAimCommand + AutoShootCommand logic into one command:

```
Constructor: SwerveSubsystem, TurretSubsystem, PivotSubsystem, FlywheelSubsystem, FeederSubsystem
Requirements: turret, pivot, flywheel, feeder
```

**execute() flow:**
1. Read limelight fiducials every 3 cycles (shared)
2. Find best target using alliance-aware filtering:
   - `DriverStation.getAlliance()` → Red: tags {10,11}, priority to 10; Blue: tags {26,27}, priority to 26
   - If priority tag is visible, aim at it; otherwise aim at the other tag in the pair
3. If no target found AND not already shooting → do nothing (stay still)
4. If target found:
   - **Turret**: PID correction on txnc (from AutoAim logic, |txnc| > 1.5 deadband, KP=0.5)
   - **Pivot**: tync → distance → lookup pivot angle (from AutoAim logic)
   - **Flywheel**: distance → lookup RPM, set target
5. When flywheel ready (±200 RPM tolerance):
   - Start feeder wheel immediately
   - After 0.3s, feeder motor joins
6. `m_isShooting` commit flag (from AutoShoot)

**end():** Stop flywheel, idle feeder, reset turret & pivot to 0, clear m_isShooting.

## Task 2: New PassShootCommand (Right Bumper)

### 2A. Rewrite `PassShootCommand.java`
**File:** `src/main/java/frc/robot/commands/PassShootCommand.java` (REWRITE)

```
Constructor: SwerveSubsystem, TurretSubsystem, FlywheelSubsystem, PivotSubsystem
Requirements: turret, flywheel, pivot
Constants: PASS_RPM, PASS_PIVOT_ANGLE (from Constants)
```

**execute() flow:**
1. Read drivebase heading: `double carHeading = drivebase.getHeading().getDegrees();`
2. Calculate turret target: `double turretTarget = -carHeading;`
3. Check turret limits: if `|turretTarget| > 90` → refuse (don't spin flywheel, don't feed)
4. If within limits:
   - Set turret to `turretTarget`
   - Start flywheel at PASS_RPM
5. Use a timer: after 0.3s of turret moving:
   - Set pivot to PASS_PIVOT_ANGLE (30°)
   - Start feeder sequence (wheel first, then motor after another 0.3s)

**end():** Stop flywheel, idle feeder, reset turret & pivot to 0.

**Phases:**
- Phase 1 (0-0.3s): Turret moves to counter-rotate, flywheel starts spinning to save time
- Phase 2 (0.3s+): Pivot goes to 30°, feeder starts when flywheel reaches target RPM
